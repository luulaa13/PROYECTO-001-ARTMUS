# PROYECTO-001-ARTMUS
Artmus – Daily UI Challenge

Artmus es una aplicación web/móvil que facilita a los usuarios descubrir museos, explorar exposiciones y programar visitas. Este proyecto forma parte de mi práctica de diseño y desarrollo, enfocándose en una experiencia completa de UX/UI, desde la idea hasta la implementación.

🎯 Alcance del proyecto

Aunque el proyecto completo incluye investigación, UX, prototipos y pruebas de usabilidad, en esta fase me estoy centrando en desarrollar la pantalla del Daily UI como una demostración funcional del diseño.

Áreas trabajadas en el proyecto:

Investigación de usuarios y benchmarking
Ideación y definición de flujo
Diseño UX y wireframes
Prototipos interactivos
Test de usabilidad
Diseño visual (UI)
Desarrollo de la pantalla Daily UI

Nota: El desarrollo completo de la app se realizará más adelante; actualmente, la implementación cubre principalmente la interfaz de la pantalla Daily UI.

💡 Idea principal

Artmus busca:

Facilitar la planificación de visitas a museos.
Ayudar a descubrir exposiciones según intereses y ubicación.
Mejorar la experiencia de exploración cultural mediante un diseño limpio y funcional.

📐 Diseño y UX
Investigación de usuarios y tendencias en apps culturales.
Creación de wireframes y prototipos interactivos.
Tests de usabilidad para validar flujo e interacciones.
Diseño visual centrado en la claridad, tipografía legible y accesibilidad.

🖥 Desarrollo
La pantalla Daily UI está implementada en React Native.
Se incluyen componentes funcionales y animaciones básicas para reflejar la experiencia de usuario.
Se utilizan buenas prácticas de desarrollo móvil para mantener la escalabilidad y consistencia del proyecto.
🔧 Tecnologías
React Native
Expo (opcional para prototipos rápidos)
JavaScript / TypeScript
React Navigation
Styled Components
